Who Created Simplistic Design?

As you might have guessed from previous pages that you might have visited on this site, Isaac Maddox is the creator of Simplistic Design. If you go to the about page on this site, you can read more about why I created this. Simply put though, I wanted a "simplified" version of Google's Material Design. So I made this.

What is Simplistic Design meant to Do?

Simplistic Design started out when I wanted a way to have a beautiful webpage, without having to code the CSS every time. It is much like Bootstrap or Material Design.

More About Isaac

I am a year-old web developer. I like to code things just for fun. They don't have to serve a purpose at all, sometimes I just sit down and code. This project, however, is one of my first long-term, constantly updating projects.
