Layout

Layout Basics

In Simplistic Design, a basic layout guideline is followed: no two elements can occupy the exact same space at the same time. This simply means that if two elements are on the same z-index, they cannot occupy the same space at the same time. For example, the aside and the article do not overlap, neither do the headers. Elements may overlap if they are on a different level. For example, the dialog box can be in the same visual space as the article, but they are not defying the laws of physics and occupying the exact same space at the exact same time.

Navigation Layout

As far as navigation goes in Simplistic Design, there is not much as far as "rules" go. All that is required is that the side navigation bar is on a higher z-index than the main navigation bar.

As you might be able to tell, there are two basic types of navigation: the main navigation bar and the side navigation bar. You can read more about these in their respectful sections.
