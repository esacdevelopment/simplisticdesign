What is Simplistic Design?

Simplistic Design is a "simplified" version of Google's Material Design. I personally have tried to use Material Design, but I found it difficult. This is not to say that I dislike it. There is a reason that I am "copying" it, after all. The only true problem that I found with Material Design was the classes. For some reason, they just didn't make sense to me. So I set out to make Simplistic Design.

How do I Use Simplistic Design?

It's simple. I will make a brief rundown here, but if you want to see the full documentation on downloading this style and script sheet, go to the Download Page. To use Simplistic Design, you do not have to recieve permission. This is an open-source library. The only thing that I ask is that you put a credit somewhere on the page. You can see the credit that is preferable to me in the footer of this page. To download the page, go to the download page, then follow the instructions. If you prefer not to download the files, you can alternatively link them directly through the internet. Please note: the file locations may change in the course of the development of Simplistic Design. The files themselves will change, if you use the online version. Visit the syntax pages to learn more about how to use the library.
