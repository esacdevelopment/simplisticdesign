Latest Release

In the latest release of Simplistic Design, I made a couple of improvements, including the select options being visible only on the available screen space, by using the overflow: auto property. I also fixed a bug with the side nav in which even a mousedown on one of the buttons would close the side nav. Obviously, I made this page, and I finalized (for now) the main pages on Simplistic Design.
