What is Simplistic Design, Again?

Simplistic Design, as you can see on the home page for Simplistic Design, is a stylesheet and a JavaScript file used to style webpages. This project is mainly for me, as I wanted a way to make webpages easily and beautifully. Simplistic Design also has a dark theme. Click this text to try that out! For a full documentation on what Material Design is, go to the about page for Simplistic Design. Now, Simplistic Design is not only to benefit me, it is also made for the people of the world. If you want to try Simplistic Design, you can download it here (keep in mind a better download system should be coming soon).

Why Should I Use Simplistic Design?

Simplistic Design is supposed to be a personalized (to me), and simpler version of Google's Material Design. In my previous experiences with this amazing product by Google, I found it hard to unserstand the class system that they had in place. Therefore, I set out on a mission to make my own Material Design but to make the class system a lot more understandable to me. For the most part, my class system follows a simple pattern: all classes begin with sd-, and then the rest of the class is what the element is. For example, the header at the top of this page has the class sd-page-header. Simple!
