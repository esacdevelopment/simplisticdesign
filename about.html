About Simplistic Design

What is Simplistic Design?

Simplistic Design is a design library created by me, Isaac Maddox. Simplistic Design is based heavily off of Google's Material Design, which I personally love.

What does Simplistic Design do?

Simplistic Design is made to allow web developers to easily make beautiful looking pages, for both computers and mobile devices. It is based on Google's Material Design. All they have to do is download the files or link them from the internet.

How to Use Simplistic Design?

If you want to use Simplistic Design, go ahead. You can download the files here, or you can link them from the internet. These instructions are also on the download page.

Other Information

Simplistic Design has been validated with the W3C CSS Validator and been approved.
