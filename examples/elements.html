Navigation

Navigation is simple. You can see all the navigation on the page. At the top, you see the main navigation bar. If you are on a computer, click the three bars in the top left. If you are on mobile or a smaller screen, click the down arrow in the top right. You can now see the side navigation bar. Also, on mobile, you can see the buttons in the navigation bar.

Dialogs

Dialogs cannot be seen by default on the page, without the developer specifying this. For more information on how a dialog works, visit the JavaScript syntax page, or the HTML syntax page.

To view a dialog, you can click the button in the bottom right of the screen. Please note: this is only the case for this page because I specified an onClick attribute for the button. This is not the default behavior of the button.
