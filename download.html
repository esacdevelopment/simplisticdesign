To download the files needed to use Simplistic Design, right-click the following links and select "Save Link As..." or "Save Target As..." in the menu that appears.
CSS File | JavaScript File
The current download is in Beta 1.6.
To add Simplistic Design to your webpage or website, copy and paste the following code into the <head> tag of your HTML file:

<link rel="stylesheet" href="{LOCATION OF STYLESHEET}">
<script src="https://code.jquery.com/jquery-2.1.0.js"></script>
<script src="{LOCATION OF SCRIPT FILE}"></script>
<link href="https://stackpath.bootstrapcdn.com/font-awesome/4.7.0/css/font-awesome.min.css" rel="stylesheet" integrity="sha384-wvfXpqpZZVQGK6TAh5PVlGOfQNHSoD2xbE+QkPxCAFlNEevoEH3Sl0sibVcOQVnN" crossorigin="anonymous">

Please note that it will be easier if you download the files into a folder inside the project folder and reference them there. If you need to use Simplistic Design for multiple projects, you can then copy and paste that folder into the other project folder. Enjoy!
